# Inbox

> [!purpose] Capture first, organize later
> Drop anything here that does not yet have a clear home. Triage at least weekly: most things should graduate to a `10-Projects/`, `20-Areas/`, or `30-Resources/` folder. If something has been sitting here for more than a month with no movement, either delete it or promote it.

**Triage checklist:**

- [ ] Does it describe a project with a goal and a deadline? → move to `10-Projects/`
- [ ] Is it an ongoing responsibility? → move to `20-Areas/`
- [ ] Is it reference material I'll want to look up again? → move to `30-Resources/`
- [ ] Is it already done / irrelevant? → delete or move to `40-Archives/`
- [ ] Still ambiguous? Leave it here for next week — but add a `triage: ?` tag.
