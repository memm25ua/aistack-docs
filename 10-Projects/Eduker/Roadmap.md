# Eduker Roadmap

Living roadmap for the Eduker product. Update `updated` whenever something material changes.

## Now (current quarter)

- [x] Repo scaffolding: Node/TS, Express, ESLint, Prettier
- [x] Placeholder home page
- [ ] SQLite schema for users, courses, lessons, quiz attempts
- [ ] Instructor: create a course (title, description, lessons)
- [ ] Learner: browse courses, take a lesson, complete a quiz

## Next

- [ ] Auth (email + magic link or OAuth — see [RFC: Auth](./RFCs/Index.md))
- [ ] Video upload / link for lessons
- [ ] Learner progress dashboard
- [ ] Email notifications (course enrolled, quiz passed)

## Later

- [ ] Mobile-friendly lessons
- [ ] Course discovery / search
- [ ] Instructor analytics
- [ ] Payments (paid courses?)

## Won't (for now)

- Live cohorts / synchronous sessions
- Authoring collaboration (multi-instructor)
- Native mobile apps

> [!tip] How to use this file
> When you finish a `Now` item, check the box and move it to a "Done" section at the bottom. When a `Next` item is being actively worked on, move it up to `Now`. The roadmap is a living artifact — keep it accurate.
