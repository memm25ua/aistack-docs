# Meetings

Meeting notes live here. One file per meeting, named `YYYY-MM-DD - <topic>.md`.

## Conventions

- Use `90-Templates/Meeting Note.md` as the starting point.
- Tag with `project/eduker` and the meeting topic.
- Always include: date, attendees, decisions, action items.
- Action items get a ``Owner Name`` wikilink and a Paperclip issue id when one exists.

> [!tip] Before the meeting
> Paste the previous meeting's "Open questions" at the top of your new note so they are visible.
