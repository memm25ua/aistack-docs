# Templates

> [!warning] Don't edit templates in place
> Always **copy** a template, then fill in the copy. Templates are the docs-of-record for note structure.

## Available templates

| Template | Use for |
| --- | --- |
| `Project Space/README.md` | The landing page for a new project space in `10-Projects/`. |
| `Area/README.md` | The landing page for a new area in `20-Areas/`. |
| `RFC.md` | A single Request for Comments. |
| `Meeting Note.md` | Meeting notes. |
| `Postmortem.md` | Incident write-up. |
| `Resource Entry.md` | A single reference page in `30-Resources/`. |

## How to use

In Obsidian, the built-in way is via the **Templates** core plugin: configure this folder as the template location, then `Insert template` will drop a copy at your cursor. From a terminal or another editor, just `cp` the file and edit.
