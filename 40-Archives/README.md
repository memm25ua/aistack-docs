# Archives

Read-only home for completed or cancelled projects, and old notes that no longer reflect how we work.

## How to archive

- Move the whole folder: `10-Projects/FinishedProject/` → `40-Archives/Projects/FinishedProject/`.
- Add a `archived: YYYY-MM-DD` frontmatter field to the folder's `README.md`.
- Add a one-line entry to the index below so future-you can find it.
